# 🚀 دليل تشغيل نظام مهيأ ERP

## ⚠️ خطوة مهمة جداً — لازم تعملها الأول

النظام محتاج تشغّل ملفين SQL في Supabase عشان يشتغل صح.

---

## الخطوة 1: شغّل الـ Migration الرئيسي

1. افتح [Supabase Dashboard](https://app.supabase.com)
2. روح لـ Project بتاعك → **SQL Editor**
3. انسخ كل محتوى ملف `supabase-migration.sql`
4. الصقه في الـ SQL Editor
5. اضغط **Run**

> ⏱️ هياخد حوالي 5-10 ثواني

---

## الخطوة 2: شغّل بيانات تجريبية

1. في نفس الـ **SQL Editor**
2. انسخ كل محتوى ملف `sample-data.sql`
3. الصقه واضغط **Run**

> ✅ ده هيضيف: منتجات، تصنيفات، موظفين، دخل، مصروفات، طلبات، عملاء، مواد خام

---

## الخطوة 3: اعمل Storage Bucket

1. روح لـ **Storage** في Supabase Dashboard
2. اضغط **New bucket**
3. اكتب الاسم: `product-images`
4. خلّي الـ **Public bucket** ✅ ON
5. اضغط **Create bucket**

---

## الخطوة 4: شغّل التطبيق

```bash
npm run dev
```

افتح `http://localhost:5173/login` وسجّل دخول بـ Supabase Auth.

## 🐛 لو في مشكلة:

### "لا توجد بيانات" في صفحة معينة؟
- تأكد إنك شغّلت `sample-data.sql`
- ريفريش الصفحة (Ctrl+R)

### Error في الـ SQL؟
- شيل الـ `ON CONFLICT DO NOTHING` من الجمل اللي فيها error
- أو شيل INSERT statements واحدة واحدة عشان تعرف فين المشكلة

### الصور مش بتظهر؟
- تأكد إنك عملت Storage Bucket اسمه `product-images`
